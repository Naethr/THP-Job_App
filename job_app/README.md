# job_app

Petite application React/Vite pour pratiquer Redux classique avec routing.

## Commandes

```bash
pnpm install
pnpm dev
pnpm run build
pnpm run lint
```

## Fonctionnalites

- Routing avec `react-router-dom`.
- Store Redux classique avec `createStore`.
- Lecture du state global avec `useSelector`.
- Mise a jour du state global avec `useDispatch`.
- Persistance minimale dans `localStorage`.
